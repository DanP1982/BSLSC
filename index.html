About this tool

This app helps Brighton Surf Life Saving Club coaches plan Saturday morning sessions for Nippers, Juniors, Youth, Seniors and Masters. It is designed to make session planning quicker, more consistent and easier to record.

It is a work in progress. The more it is used, the more insight we will have to make adjustments and improvements. Your feedback is genuinely valued — please reach out to Dan P with any thoughts.

How to use it
1. Weather tab — enter the date, start time, coach and safety cover. Tap ⛅ Populate Weather to auto-fill conditions, and ⛅ Auto-fill in the Tides section to fetch Brighton Marina tide times for the session date. Both require an internet connection. Manual tide lookup links are provided as a backup.
2. Session tab — select a group from the dropdown. Use + Add on each block to browse the activity library, or tap ✦ to get AI suggestions for that block. Once your warm up, theory and main set are set, tap ✦ Generate to auto-create session aims — then edit them to suit.
3. Library tab — browse and search activities filtered to the selected group. Activities are categorised as Warm Up, Theory or Main Set.
4. Schedule tab — set start times for each group. These feed through automatically to the session time when you switch groups.
5. Print / Save — tap 🖨 to print the current group's session plan as a CP1-format PDF, save to Photos, or save the full plan as a file to reopen later.
⚠️ AI disclaimer

Some content in this tool is generated by artificial intelligence — including weather forecasts, session aim suggestions and activity recommendations.

AI can make mistakes. It is the responsibility of the coach to review all details before using them in a session, including auto-populated weather and conditions data. Always apply your own judgement and refer to live conditions on the day.

💬 Share your feedback

If you use a free-text field to describe an activity or session aim that works well, and you think it would be useful to add to the library for others, please share it with Dan P.

Equally, if something doesn't work as expected, an activity description is wrong, or you have ideas for improvement — please get in touch. The tool will get better with use.
